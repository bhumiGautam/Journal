# Journal Analyzer

This repository contains a full-stack journaling and analysis application.

## Project Structure

- `backend/` - Express server, database, and AI integration.
- `frontend/` - React/Vite client interface.

## Running Locally

### Backend

1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Notes

- The backend includes routes for journal entry CRUD and analysis.
- The frontend uses a simple React UI to create entries and view insights.
